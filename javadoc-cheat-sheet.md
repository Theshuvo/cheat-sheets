# Javadoc Style Guide Cheat Sheet

## Code `{@code }`

- `{@code arg}`: Refer to Method Parameter [(from stackoverflow)](http://stackoverflow.com/q/1667212/1833118).

  ```
  /**
   * If {@code arg} is null, ....
   * @param arg
   */
  void foo(String arg) {...}
  ```

## Link `{@link }`

- `{@link ClassName}`: Link to a Class
- `{@link #field}`: Link to field of *this* class
